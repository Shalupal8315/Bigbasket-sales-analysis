# BigBasket Sales Data Analysis & Dashboard

## Project Overview

This project focuses on analyzing the BigBasket product dataset and creating an interactive Power BI dashboard to understand sales patterns, product performance, and category insights.

## Project Workflow

### 1. Data Cleaning

First, the raw dataset was cleaned to remove missing values, duplicate records, and inconsistent data. This step ensured that the dataset was accurate and ready for analysis.

### 2. Data Analysis

After cleaning the dataset, exploratory data analysis was performed using Python. The analysis process and insights are documented in the **analysis.ipynb** file.

### 3. Data Visualization

Using the cleaned dataset and analysis results, an interactive **Power BI dashboard** was created to visualize important insights such as:

* Top selling products
* Sales by category
* Discount distribution
* Product ratings

## Tools & Technologies Used

* Python (for data cleaning and analysis)
* Jupyter Notebook
* Power BI
* CSV Dataset

## Key Insights

* Identified top-performing product categories
* Analyzed product pricing and discounts
* Compared ratings across different products
* Visualized category-wise sales trends

## Project Files

**dataset.csv** – Raw dataset
**cleaned_dataset.csv** – Cleaned dataset used for analysis
**analysis.ipynb** – Data analysis using Python
**dashboard.pbix** – Power BI dashboard file
